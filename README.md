## 项目简介

本项目利用 **react18.2** + **json-server** 做了一个后台管理系统。

包含：

- 用户管理
- 权限管理
- 站内信审核管理
- 站内信发布管理

其中涉及到react-router V6.0的使用以及一些权限控制等内容。



**本项目很简陋，有很多需要改进的地方**，但是可以作为为二次开发的基础版本。

## 项目运行


安装依赖：

```
yarn
```

启动json-server：

```
yarn mock
```

启动项目:

```
yarn start
```
